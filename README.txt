# AlansCinematics site

Files:
- index.html: page structure and content
- style.css: site styling
- script.js: interactive behavior
- images/: all image assets extracted from the original HTML

How to preview:
1. Open this folder in VS Code.
2. Open index.html in a browser, or use Live Server.

How to publish:
1. Upload this whole folder to a GitHub repo.
2. Use GitHub Pages, Netlify, or Cloudflare Pages.
